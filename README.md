Save and access links instantly.

With one click, save your current tab or any URL for quick access later.

A lightweight Chrome extension designed to help you collect, manage, and revisit important links.

# How to Use

Paste a URL into the input field and click Save Input,
or click Save Tab to save your current open tab.

Saved links appear at the bottom of the extension.

Click any saved link to open it in a new window.

Double-click Delete All to remove all saved links.
